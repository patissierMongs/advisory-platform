# Chip

Removable token (e.g. extracted CVE codes).

**Group:** Data Display  ·  **Variants shown:** removable tokens (extracted CVEs)

Available from the design system global as `AdvisoryDS.Chip`.

## Props

| prop | type | notes |
|---|---|---|
| `onRemove` | `() => void` | shows × button when set |
| `mono` | `boolean` | monospace text |

const { Chip } = AdvisoryDS;
