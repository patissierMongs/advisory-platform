# Card

The base white panel surface every section sits on.

**Group:** Layout  ·  **Variants shown:** Base white surface

Available from the design system global as `AdvisoryDS.Card`.

## Props

| prop | type | notes |
|---|---|---|
| `padding` | `number|string` | inner padding, default 20 |
| `flush` | `boolean` | drop border+radius when nesting |

## Usage

```jsx
const { Card } = AdvisoryDS;
```

See `Card.jsx` for a complete, rendered example.

## Notes

Compose freely — put headings, tables, stats inside.
