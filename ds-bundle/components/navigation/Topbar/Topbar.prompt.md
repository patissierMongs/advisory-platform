# Topbar

White app header bar with title + right-aligned actions.

**Group:** Navigation  ·  **Variants shown:** page header with status + user

Available from the design system global as `AdvisoryDS.Topbar`.

## Props

| prop | type | notes |
|---|---|---|
| `title` | `ReactNode` |  |
| `subtitle` | `ReactNode` |  |
| `actions` | `ReactNode` | status pills, Avatar, buttons |

## Usage

```jsx
const { Topbar } = AdvisoryDS;
```

See `Topbar.jsx` for a complete, rendered example.

## Notes

Height 60px, sits above the scrolling content area.
